Перед загрузкой выберите:

  • Период, за который будут загружены курсы валют.
  • Валюты, для которых будут загружены курсы за указанный период.

Затем нажмите кнопку «Загрузить курсы». Загруженные курсы сохраняются в регистре сведений «Курсы валют».

Для загрузки курсов валют из файла необходимо выбрать команду: Все действия - Загрузить курсы из файла...

Для настройки автоматической загрузки курсов валют с веб-сайта Национального Банка РК необходимо выбрать команду: Все действия - Автоматическая загрузка курсов...

При загрузке курсов валют с веб-сайта Национального Банка РК действуют настройки прокси сервера на сервере 1С:Предприятие. Настройки прокси-сервера на клиенте не учитываются. Параметры прокси-сервера на сервере 1С:Предприятие можно задать в настройках программы.

См. также: Валюты, Курсы валют, Загрузка курсов валют из файла, Автоматическая загрузка курсов валют, Параметры прокси-сервера.
