Справочник "Доходы"

Справочник «Доходы» предназначен для хранения перечня видов дохода организации.

Кнопка «Показывать коды строк Декларации по налогу на прибыль» панели инструментов позволяет настроить отображение кодов строк Декларации в форме списка для каждого элемента справочника.

Для каждого элемента присутствует возможность указания вида деятельности, с которым связано получение данного вида дохода.

Для отражения вида дохода в регламентированной налоговой отчетности используются следующие данные:

  • На закладке «Отражение в строках Декларации» указываются коды строк декларации для регламентированного отчета «Форма 100.00. Декларация по корпоративному подоходному налогу».
  • На закладке «Отражение в упрощенной декларации» указываются принадлежность данного вида дохода к налогооблагаемому доходу. Если признак «Является объектом налогообложения» установлен, то доход включается в подсчет общей суммы налогооблагаемого дохода, полученного за налоговый период, в регламентированный отчет «Форма 910.00. Упрощенная декларация для субъектов малого бизнеса».
