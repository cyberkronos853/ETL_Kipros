Справочник «Банки»

Справочник «Банки» предназначен для хранения информации о банках, в которых имеются расчетные счета организации, контрагентов или физических лиц, с которыми у организации возникают расчеты в безналичном порядке. В справочник заносится информация о наименовании банка, БИК, контактная информация (местонахождение и телефон) и реквизиты электронного платежного поручения.

Для банка можно указать контрагента, в качестве которого может быть элемент справочника «Контрагенты» в виде банка с таким же названием и реквизитами, только с более детальной контактной информацией, реквизитах, счетах и договорах.

Список банков можно заполнять автоматически, используя «Общий классификатор банков». Классификатор открывается из формы списка по кнопке «Подбор». Добавление банка в справочник осуществляется двойным щелчком по необходимой строке с банком. Автоматически заполняется наименование банка, БИК, адрес и код в платежной системе. Контактную информацию необходимо заносить самостоятельно. Если при выборе значения из классификатора элемент с таким БИК уже есть в справочнике, то конфигурация выдаст предупреждающее сообщение, если такого элемента нет, то будет сформирована новая форма элемента с заполненными параметрами.

Справочник является иерархическим, что позволяет пользователю разделять элементы по группам. Для элемента справочника обязательным для заполнения является поле «Наименование». Если указанное поле не заполнено, система не позволит записать элемент справочника.
